新時代判讀力

Post modified 2017-12-20 | In 閱讀 | 0 comments
| 1,865

在這紛亂的資訊時代，有多少資訊是相對正確，或者是能夠精準無誤的將一件事物精準描述並讓閱聽人能夠接收到正確的資訊？講跟生活較為相關的食物與身體健康保健，我們能夠相信新聞告訴我們的資訊是正確的，還是當笑話看看，不過只是廠商的業配文？相對正確

簡介

介紹一下這本書的來歷，他是泛科學的團隊，叫科學新聞解剖室主要在分析新聞中出現的各種假借科學名義而造成人們誤會的奇怪文章，並判斷內容到底有幾分真幾分假。

說真的，舉個例子，為什麼歐陽XX可以這麼的占新聞版面，她們，講難聽點，乾我啥事，我不過是市井小民，不需要告訴我有錢人家的小孩是怎麼過日子的。（扯遠了…

這就算了

最糟糕的，是給予觀眾錯誤的概念。原本新聞應該是要幫助人們用較為簡單的方式得到「正確」資訊，但…通常看到的都是錯誤的。（亦或是過去台灣媒體造就出來的奇怪習慣。這裡就跟大家分享書中提到的幾個奇怪現象吧！

過於驚悚的新聞標題

這個不論是不是在科學新聞上，驚悚的標題代表著可觀的點閱率，也代表著有更多的機會可以賺錢（點閱率越高，廣告效益越好摟！）。所以說，常常在新聞中看到很多不分青紅皂白的選擇性報導採訪內容。例如在2014年6月4日報導的「十大恐怖外食」就很有種不分青紅皂白的將沒有經過精確實驗過後所得到的結論，硬是加在食物上面。

最近的新聞，例如這種吸睛標題「定期上教堂的人 研究：比較長壽」雖然內容沒什麼用（標題與內容不符。最後提到有反面的意見，「其實是因為常上教堂的人心情放鬆，行動比較方便，所以才會導出這樣的研究結果。」所以這個新聞真的沒什麼用處阿…….。

資料來源不明的定論

我不懂的地方在於，一份學術期刊要發表，要經過長時間的研究與討論，並與同領域的人互相審查之後，才能夠經歷千辛萬苦投稿。

但是！新聞時常都會先發文之後再跟你說對不起我錯了我下架。喔，這件事情也不只發生在新聞上拉。不過在學術期刊中，就算審核過，也不乏有寫作錯誤的部份，何況是基本上都是文組出生而沒有經過一連串的科學訓練過的新聞媒體記者呢？

請理性溝通，我的意思不是要戰文組理組，而是想知道為什麼不能彼此互相尊重？政治正確的使用語言與現在的台灣社會似乎還有一大段的路要走。

轉移焦點

美女正妹，也是另外一個使新聞版面變奇怪的部份。「十大恐怖外食」新聞中所提到的營養師，好吧，她真的是一位營養師，但是…到底跟正妹有什麼關係？雖然男生喜歡看正妹但這種健康新聞跟外表結合的奇怪連結到底是怎麼一回事呢？我真的百思不得其解。

另外也會引用「英國研究」。
喔對，這個英國研究其實是來自於「每日郵報」，不知道為什麼新聞很喜歡引用每日郵報呢？是因為翻譯簡單嗎還是……。

內容農場

這也是個很夭壽的部份，有時候在臉書上會看到大家發文，然後發的文章是內容農場的文章，根據wiki上的說明

內容農場（Content Farm）是指圖謀網路廣告等商業利益的專業公司，以取得網路流量為主要目標。內容農場用各種合法、非法之手段大量、快速的生產品質不穩定的網路文章，會針對熱門搜索關鍵字用人工或機器製造大量網站內容的手法欺騙搜索引擎，使他們製造的網頁能夠優先出現在搜索結果的前段而提高點閱率及滿足客戶搜尋引擎優化需求，或是藉大量的點閱流量圖謀網路廣告收益。

然後婆婆媽媽們便會大量轉貼這類的文章，重點是「有趣的標題」、「似是而非卻沒有意義的內容」。但為什麼大家還是會瘋狂的轉貼呢？

也許這就是人性吧，什麼東西會紅，代表人們對這個主題有興趣，但是有時候可能是上面有錢的人在操縱，而下面接收訊息的人卻沒有足夠的判斷力去了解自己接收到的訊息究竟是否為真，這樣下來的結果，必然會造成資訊的不對稱與錯誤的資料訊息。

忽略過程

這不僅僅是新聞科學會犯下的錯誤，就算是在學術界，時常也會因為自己研究的盲點，而犯下忽略過程，只注重自己看到的高度相關性。

喔我這裡說的是自己看到的，也就是說，這個不是真的相關。舉個吧，例如這個：減肥光算卡路里 恐會愈減愈肥很明顯奇怪吧？這兩個可能有關，但他的連結標題直接下的很像是直接相關。雖然多了個恐字，但這種寫作方式是否會讓人很想點進去看到底是不是這樣呢？

文章中也沒有提到，到底這樣的資料是怎麼樣子來的，還是我們應該要尊重權威？

感想

這本書是個起點，也是一個警惕。為什麼會有這種書的出現？或者是說為什麼會有人想要做這種書？代表台灣的媒體已經不再是一個可以準確提供正確資訊的平台，而是要靠著其他人用其他的方式來進行思想的辯證。如果有興趣的話其實可以看看網路上泛科學的文章。也許看個一兩個月，會發現原來台灣媒體所報導的新聞內容，很多都是速食與資本主義造就出來的結果。但若是要好好的判斷一件事物的對錯，也許好好學習理則學的內容（不知道為什麼這種重要的判讀能力在高中或是大學都沒有教導，而是要自己想辦法學習），才能夠減少犯錯的機會。從另外一個觀點來看，也許人類的發展就應該如此，簡易的判斷方式，簡易的二分法，使人類在短時間內簡單的去判斷事情的對錯，在生死交關的時候特有用。一個人都要死了，還需要判斷什麼是非對錯？先好好活下去才是重點。

好好鍛鍊判斷力吧！
